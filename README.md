# SAMURAI DUEL

Пиксельный файтинг про двух самураев. Автор и гейм-директор — Петя.
Один самурай — игрок, второй — честный ИИ (он видит только позиции
и состояния на экране, ввод игрока не читает).

**Играть:** https://zinchenko-denis.github.io/samurai-duel/
(или просто открыть `index.html` в браузере — интернет не нужен)

## Управление

| Действие      | Клавиши          |
|---------------|------------------|
| Шаг влево/вправо | ← → или A D   |
| Удар ногой    | Z или J          |
| Подножка (сбивает с ног) | X или K |
| Сюрикен       | C или L          |
| Старт / реванш | Enter           |

Матч — до двух выигранных раундов.

## Как устроен код

Всё в одном файле `index.html`, секции подписаны:

- **НАСТРОЙКИ** — имена бойцов, здоровье, урон, сложность ИИ. Меняй смело.
- **СПРАЙТЫ** — персонажи нарисованы буквами: каждая буква — цвет,
  точка — прозрачно. Сетка 32×30, масштаб ×4.
- **АРЕНА** — локация «Храм на закате». Вторая локация = ещё один
  такой объект.
- **ВВОД** — виртуальные кнопки; клавиатура и ИИ пишут в них одинаково.
- **БОЕЦ** — конечный автомат состояний. У атак есть кадры
  startup / active / recovery — как в настоящих файтингах.
- **ИИ** — тоже автомат: держит дистанцию, издали кидает сюрикен,
  вблизи бьёт или делает подножку.
- **ИГРА** — раунды, полоски здоровья, надписи РАУНД / БОЙ! / K.O.

## Как редактировать спрайты

Найди в `index.html` блок `const RAW` и меняй буквы в строках:

```
o = контур     h = волосы    w = повязка   s = кожа
a = броня      b = тёмная броня   c = светлая броня
g = пояс       p = штаны     k = обувь
```

Например, чтобы дать самураю усы — добавь пару `h` на строке рта.
Сохрани файл, обнови страницу — изменения сразу видны.
